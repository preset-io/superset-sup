Manage charts

## Usage

```bash
sup chart [COMMAND] [OPTIONS]
```

## Commands

| Command | Description |
|---------|-------------|
| `list` | List charts in the current or specified workspace |
| `info` | Show detailed information about a chart |
| `sql` | Get the compiled SQL query that powers a chart |
| `data` | Get the actual data results from a chart |
| `pull` | Pull chart definitions from Superset workspace to local filesystem |
| `push` | Push chart definitions from local filesystem to Superset workspace |
