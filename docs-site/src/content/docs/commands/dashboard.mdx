Manage dashboards

## Usage

```bash
sup dashboard [COMMAND] [OPTIONS]
```

## Commands

| Command | Description |
|---------|-------------|
| `list` | List dashboards in the current or specified workspace |
| `info` | Show detailed information about a dashboard |
