Manage databases

## Usage

```bash
sup database [COMMAND] [OPTIONS]
```

## Commands

| Command | Description |
|---------|-------------|
| `list` | List all databases in the current or specified workspace |
| `use` | Set the default database for SQL queries |
| `info` | Show detailed information about a database |
