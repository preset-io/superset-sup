Manage datasets

## Usage

```bash
sup dataset [COMMAND] [OPTIONS]
```

## Commands

| Command | Description |
|---------|-------------|
| `list` | List datasets in the current or specified workspace |
| `info` | Show detailed information about a dataset |
| `export` | Export dataset(s) to YAML files |
