Manage asset ownership

## Usage

```bash
sup ownership [COMMAND] [OPTIONS]
```

## Subcommands

| Command | Description |
|---------|-------------|
| `sup ownership pull` | Pull asset ownership to a YAML file |
| `sup ownership push` | Push asset ownership from a YAML file |
