Manage row-level security

## Usage

```bash
sup rls [COMMAND] [OPTIONS]
```

## Subcommands

| Command | Description |
|---------|-------------|
| `sup rls pull` | Pull RLS rules to a YAML file |
| `sup rls push` | Push RLS rules from a YAML file |
