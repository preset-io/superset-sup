Test themes and colors

## Usage

```bash
sup theme [COMMAND] [OPTIONS]
```

## Commands

| Command | Description |
|---------|-------------|
| `colors` | Test ASCII logo with different green options using hex colors for maximum distinction |
| `palette` | Test the full color palette |
| `banner` | Test the full banner with current branding |
