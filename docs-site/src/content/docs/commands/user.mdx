Manage users

## Usage

```bash
sup user [COMMAND] [OPTIONS]
```

## Subcommands

| Command | Description |
|---------|-------------|
| `sup user list` | List all users in the current or specified workspace |
| `sup user info` | Show detailed information about a user |
