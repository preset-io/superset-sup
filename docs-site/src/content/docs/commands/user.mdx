Manage users

## Usage

```bash
sup user [COMMAND] [OPTIONS]
```

## Commands

| Command | Description |
|---------|-------------|
| `list` | List all users in the current or specified workspace |
| `info` | Show detailed information about a user |
| `export` | Export users, roles, and permissions to files |
| `import` | Import users, roles, and permissions from files |
