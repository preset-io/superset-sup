Manage workspaces

## Usage

```bash
sup workspace [COMMAND] [OPTIONS]
```

## Commands

| Command | Description |
|---------|-------------|
| `list` | List all available workspaces |
| `use` | Set the default workspace for current session |
| `info` | Show detailed information about a workspace |
| `set-target` | Set the import target workspace for cross-workspace operations |
| `show` | Show current workspace context including source and import target |
