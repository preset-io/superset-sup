## Quick Example

```bash
# Set your workspace
sup workspace use 123

# Pull your charts
sup chart pull --mine

# Run SQL queries
sup sql "SELECT COUNT(*) FROM users"

# Push to another workspace
sup chart push --workspace-id 456
```
